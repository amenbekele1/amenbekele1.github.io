# 👨‍💻 Amen Bekele — Personal Resume Website

Personal portfolio and resume website for **Amen Bekele**, Infrastructure & Operations Engineer at GN Group (GN Audio), Warsaw.

🌐 **Live site:** [amenbekele1.github.io](https://amenbekele1.github.io)

---

## About

A single-page personal resume site built from scratch with vanilla HTML, CSS, and JavaScript. Designed with a dark terminal aesthetic to reflect a technical background in enterprise infrastructure.

---

## Sections

- **Hero** — Animated typing effect, professional intro, terminal-style profile block
- **Skills** — Categorized tech stack across virtualization, cloud, networking, systems, and security
- **Experience** — Timeline of roles at GN Group and Mindray
- **Projects** — Case studies including Project Pearl, SFTP RCA, ISO 27001, Veeam, and more
- **Certifications** — Including verified Cisco CCNA (Credly badge)
- **Contact** — Email, LinkedIn, and contact form

---

## Tech Stack

| Layer | Details |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, grid, flexbox) |
| Scripting | Vanilla JavaScript |
| Fonts | JetBrains Mono, Syne (Google Fonts) |
| Hosting | GitHub Pages |

## Features

- Dark / Light mode toggle
- Animated typing effect in hero

- Embedded Credly certification badge
